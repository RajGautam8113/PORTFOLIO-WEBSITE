# 🚀 Modern Portfolio Website

A sleek, responsive, and high-performance personal portfolio built with the latest web technologies. This project showcases my professional journey, technical skills, and featured projects.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Shadcn/UI](https://ui.shadcn.com/)
- **Deployment:** [Vercel](https://vercel.com/)

## ✨ Features

- **Responsive Design:** Optimized for all screen sizes (Mobile, Tablet, Desktop).
- **Dynamic Projects:** Showcasing work with detailed descriptions and tech stacks.
- **Modern UI:** Clean aesthetic using Tailwind and Shadcn components.
- **Fast Performance:** Server-side rendering and image optimization via Next.js.
- **Contact Form:** Integrated for direct professional inquiries.

## 🚀 Getting Started

First, run the development server:

```bash
pnpm install
# or
npm install

# Then start the server
pnpm dev
# or
npm run dev
Open http://localhost:3000 with your browser to see the result.

📁 Project Structure
/app: Main application routes and pages.

/components: Reusable UI components.

/lib: Utility functions and helper scripts.

/public: Static assets like images and fonts.

/styles: Global CSS and Tailwind configurations.
```
